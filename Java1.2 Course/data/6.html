Ternary Operator: 

x = a ? b : c

The above line means:
if a is true then x=b;
if a is false then x=c;

b and c have to be primitives.

Example:
  double i=5, j;
  j = (i>5) ? 6.0 : 4;

In this case j turns out to be 4.0, because j is upgraded to a double due
to the 6.0. Therefore, when the 4 is moved into j it becomes a double.


Assignment Operator: 

ie. the '=' operator

Example:
int i=2;
int j,k,l;
j = k = l = i; 

The above is completely legal because i has been assigned already.
So now we move from right to left, ie i --> l --> k --> j

Variables: 

Local variable (or automatic variable):
* Defined within a method
* Has to be initialised explicitly (ie by user)
* Exists only for the duration of the method

* Declared in a class

this and super: 

NOTE: this and super are never used in the main method
non-static variable 'this' cannot be referenced from a static context

Keywords:
'this', 'super'

	Parent1
        /    \
       /      \
    Child1   Child2

If you are currently in Child1 and you are trying to access a variable defined
in Parent1, eg. var1, then you use the keyword 'super'
If you are trying to access something within Child1 and you are in Child1, then
you use the keyword 'this'.


Example:
class ThisSuper extends SuperClass {  // **1**
 String s = "this";
 public static void main(String[] args) {
  new ThisSuper();
 }
 ThisSuper() {  // **2**
  display(this.s);
  display(super.s);
  this.display(s);
  super.display(s);
 }
 void display(String s) {
  System.out.println("this: "+s);
 }
}

class SuperClass {  // **3**
 String s = "super";
 void display(String s) {
  System.out.println("super: "+s);
 }
}

Output:
this: this
this: super
this: this
super: this


**1** You are defining the class ThisSuper and saying that 
it is the child of SuperClass

**2** ThisSuper seams to be also the name of a method inside the class,
its actually a constructor. The job done by this constructor can also
be achieved by a method.

**3** ThisSuper is the parent class


NOTE: For applications, there is only one MAIN method. 
Since the main method for the above program is defined 
in the ThisSuper class, you will compile it as ThisSuper.java

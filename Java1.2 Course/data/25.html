Modifying a String: 

subString()
String subString(int StartIndex)
String subString(int StartIndex, int endIndex)

Example:
String st = "Amir Khan";
String ss1 = st.subString(5);  <-- this returns "Khan"
String ss2 = st.subString(2,6);  <-- this returns "ir k"


concat()
In order to concatenate 2 strings we just put the + sign between them. When we do this 
the concat method is called by Java:
String ss3 = st.concat("cd");  <-- returns "Amir Khancd"


trim()
String st = "   Asif  Aman  ";
String ss1 = st.trim();  <-- this returns "Asif  Aman", we got rid of spaces before & after


StringBuffer class methods: 

Example:
String st = "abc";  <-- new String Object is created
String st = st.toUpperCase();  <-- new String Object is created and "ABC" is moved into it
				   Previous st Object is collected by the GC
String st  = st + "de"; <-- new String Object created and "ABCde" moved in
This can also be written as: String st = new StringBuffer().append(st).append("de").toString();


void ensureCapacity(int minimumCapacity) 
Ensures that the capacity of the buffer is at least equal to the specified minimum.

StringBuffer append()
Appends the string representation to the string buffer.


Example1:
// StringBuffer length vs. capacity.
class StringBufferDemo {
  public static void main(String args[]) {
    StringBuffer sb1 = new StringBuffer("Hello");
    System.out.println("buffer = " + sb1);
    System.out.println("length = " + sb1.length());
    System.out.println("capacity = " + sb1.capacity());

    sb1.append("1234567890123456");
    System.out.println("buffer = " + sb1);
    System.out.println("length = " + sb1.length());
    System.out.println("capacity = " + sb1.capacity());


    sb1.append("7");
    System.out.println("buffer = " + sb1);
    System.out.println("length = " + sb1.length());
    System.out.println("capacity = " + sb1.capacity());

    sb1.append("1234567890123456789012");
    System.out.println("buffer = " + sb1);
    System.out.println("length = " + sb1.length());
    System.out.println("capacity = " + sb1.capacity());

    sb1.append("3");
    System.out.println("buffer = " + sb1);
    System.out.println("length = " + sb1.length());
    System.out.println("capacity = " + sb1.capacity());
  }
}

Example2:
// Demonstrate delete() and deleteCharAt()
class deleteDemo {
  public static void main(String args[]) {
    StringBuffer sb = new StringBuffer("This is a test.");

    sb.delete(4, 7);
    System.out.println("After delete: " + sb);

    sb.deleteCharAt(0);
    System.out.println("After deleteCharAt: " + sb);
  }
}

Output2:
After delete: This a test.
After deleteCharAt: his a test.

Nested Loops: 

NOTE: There is no limit on the number of nested loops you can have.

Example1:
class Nested {
  public static void main(String args[]) {
    int i, j;

    for(i=0; i<10; i++) {
      for(j=i; j<10; j++)
        System.out.print(".");
      System.out.println();
    }
  }
}

continue statement: 

for (int i=0; i<10; i++) {
  ---------
  ---------
  if (i==6) continue;
  ---------
  ---------
}

If i is 6, then code after this line is not executed, and the loop goes to its
next increment. A break statement takes you out of the loop, but continue keeps
you inside the loop.


Example1:
class Continue {
  public static void main(String args[]) {
    for(int i=0; i<10; i++) {
      System.out.print(i + " ");
      if (i%2 == 0) continue; // when this is false then the next line is executed
      System.out.println("");
    }
  }
}

Output1:
0 1   <--- 1%2 is not zero thus the line System.out.println(""); is executed
2 3
4 5
6 7
8 9




Labels (flow controls): 

Its like an alias given to a block of statements.

first: {  
	------
	------
	second: {  <-- nested block label
		 ------
		 ------
	}
	------
}


Example1:
class Break {
  public static void main(String args[]) {
    boolean t = true;

    first: {
      second: {
        third: {
          System.out.println("Before the break.");
          if(t) break second; // break out of second block
          System.out.println("This won't execute");
        }
        System.out.println("This won't execute");
      }
      System.out.println("This is after second block.");
    }
  }
}

Output1:
Before the break.
This is after second block.


NOTE: This kind of break can only be applied to the current block/loop or its
outer block/loop. If you tried to break a block which is not a part of the current
block, it will give the error 'Undefined label' at compile time. 
Labels can also be given to loop blocks.

Example2:
class ContinueLabel {
  public static void main(String args[]) {
outer: for (int i=0; i<10; i++) {
         for(int j=0; j<10; j++) {
           if(j > i) {
             System.out.println();
             continue outer;
           }
           System.out.print(" " + (i * j));
         }
       }
       System.out.println();
  }
}

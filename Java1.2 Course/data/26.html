Introduction to Applets: 

First used for the web, executed on the browser

To view an applet there are 3 techniques:

1) View applet in browser
<applet codebase="." code=SimpleApplet.class width=460 height=50> </applet>

Example1:
import java.awt.*;
import java.applet.*;

public class SimpleApplet extends Applet {
  public void paint(Graphics g) {
    g.drawString("Abdul's Simple Applet", 20, 20);
    g.draw3DRect(5, 5, 200, 50, true);
    int[] Xarray = {20,30,40};
    int[] Yarray = {40,20,40};
    g.drawPolygon( Xarray, Yarray, 3);
  }
}

/*
<applet code="SimpleApplet.java" width=200 height=60>
</applet>
*/

> javac SimpleApplet.java

init() method:
Called by the browser or applet viewer to inform this applet that it has been loaded 
into the system


destroy()
Called by the browser or applet viewer to inform this applet that it is being reclaimed 
and that it should destroy any resources that it has allocated

setForeground(Color c)
Defined in the init method. Sets the foreground color of this component.
eg. setForground(Color.white);
Also Color red = new Color(256,0,0);  <-- ie. (Red, Green, Blue)


Applet structure: 

// An Applet skeleton.
import java.awt.*;
import java.applet.*;
/*
<applet code="AppletSkel" width=300 height=100>
</applet>
*/

public class AppletSkel extends Applet {
  // Called first.
  public void init() {
    // initialization
  }

  /* Called second, after init().  Also called whenever
     the applet is restarted. */
  public void start() {
    // start or resume execution
  }

  // Called when the applet is stopped.
  public void stop() {
    // suspends execution
  }

  /* Called when applet is terminated.  This is the last
     method executed. */
  public void destroy() {
    // perform shutdown activities
  }

  // Called when an applet's window must be restored.
  public void paint(Graphics g) {
    // redisplay contents of window
  }
}


Example:
/* A simple applet that sets the foreground and
   background colors and outputs a string. */
import java.awt.*;
import java.applet.*;
/*
<applet code="Sample" width=300 height=50>
</applet>
*/

public class Sample extends Applet{
  String msg;

  // set the foreground and background colors.
  public void init() {
    setBackground(Color.cyan);
    setForeground(Color.red);
    msg = "Inside init( ) --";
  }

  // Initialize the string to be displayed.
  public void start() {
    msg += " Inside start( ) --";
  }

  // Display msg in applet window.
  public void paint(Graphics g) {
    msg += " Inside paint( ).";
    g.drawString(msg, 10, 30);
  }
}
